This directory contains the files require to do Lab 2

Files:
	lab2.pdf:          The lab writeup
	lab2-solution.pdf  The lab solution

	files/{cyc3.cnf,cyc3-dct.lrat,cyc3-bkt.lrat}: Sample Boolean formula and its LRAT proofs

	generators/mdparity.py: Code to generate formulas for minimum disagreement parity problem (MDP)
		    Run with the -h flag to see the options

        generators/mdpcheck.py: Checker for solutions to MDP generated by a SAT solver

	generators/writer.py: Support code for the generator.  You need not modify this file
